# 通訊錄管理系統 (Contacts Management System)

## 功能特色 (Features)

### 基本功能
- **新增聯絡人** - 可新增學號、班級、姓名、電話資訊
- **修改聯絡人** - 可修改已選擇的聯絡人資訊
- **刪除聯絡人** - 可刪除已選擇的聯絡人
- **清空輸入** - 快速清空所有輸入欄位
- **匯入/匯出** - 支援CSV檔案的匯入與匯出

### 進階功能
- **即時搜尋** - 可依學號或姓名搜尋聯絡人
- **雙擊編輯** - 雙擊表格項目即可載入資料準備編輯
- **狀態顯示** - 視窗標題顯示總資料數、顯示數量及選擇狀態
- **表格排序** - 點擊欄位標題可進行排序
- **輸入驗證** - 所有欄位必須填寫才能新增
- **重複檢查** - 防止學號重複

### 使用者介面優化
- **快捷鍵支援**
  - Alt+A: 新增
  - Alt+E: 修改
  - Delete: 刪除
  - Ctrl+L: 清空輸入
  - Ctrl+S: 匯出
  - Ctrl+O: 匯入
  - Ctrl+Q: 結束
- **視覺改善**
  - 交替行顏色便於閱讀
  - 群組框架分區清晰
  - 輸入提示文字
  - 按鈕工具提示
- **確認對話框**
  - 刪除前確認
  - 結束前確認
  - 匯入前確認（當有資料時）

## 如何使用

1. **新增聯絡人**
   - 填寫所有欄位（學號、班級、姓名、電話）
   - 點擊「新增」按鈕或按 Alt+A
   - 輸入欄位會自動清空

2. **修改聯絡人**
   - 方法一：點選表格中的聯絡人，修改輸入欄位後點擊「修改」
   - 方法二：雙擊表格中的聯絡人，系統會自動載入資料到輸入欄位

3. **刪除聯絡人**
   - 點選要刪除的聯絡人
   - 點擊「刪除」按鈕或按 Delete 鍵
   - 確認刪除對話框

4. **搜尋聯絡人**
   - 在搜尋框輸入學號或姓名的部分關鍵字
   - 系統會即時過濾顯示符合的結果

5. **匯出資料**
   - 點擊「匯出」按鈕
   - 選擇儲存位置和檔名
   - 資料會以 CSV 格式儲存

6. **匯入資料**
   - 點擊「匯入」按鈕
   - 選擇要匯入的 CSV 檔案
   - 系統會載入所有聯絡人資料
